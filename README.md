## Падежи

1. Номинатив (ном.)
2. Генитив (ген.)
3. Датив (дат.)
4. Акузатив (ак.)
5. Вокатив (вок.)
6. Инструментал (инстр.)
7. Локатив (лок.)

| Падеж | Питање |
| Номинатив | Ко? Шта? |
| Генитив | Кога? Чега? Од кога? Од чега? Чији? |
| Датив | Коме? Чему? |
| Акузатив | Кога? Шта? Куда? |
| Вокатив | Хеј! |
| Инструментал | С ким? Са ким? Чим? С чим? Чиме? |
| Локатив | Где? О коме? О чему? На чему? |
